Visualizing Wildfire Risks to Homes Across Florida.

by Misha Vaid

This is an exploratory map that has not been vetted by scientific experts. It was made as an assignment for the Points Unknown Web Mapping Class in June 2022.

Data source: U.S. Forest Services's Wildfire Risk to Communities accessed through The Brown Institute. All code is available on GitHub.

But which counties are more at risk?

One way to measure Wildfire risk, is to look at every county's total number of exposed Housing Units as a percentage of the total number of units.

But what exactly does this metric mean? According to the USDA wesbite, "Housing Unit Exposure is the expected number of housing units within a pixel potentially exposed to wildfire in a year. This is a long-term annual average and not intended to represent the actual number of housing units exposed in any specific year."
